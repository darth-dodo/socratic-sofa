# Topic Library Guide

Welcome to the Socratic Sofa topic library - a curated collection of 100 philosophical questions designed to spark deep inquiry and critical thinking.

## Overview

The topic library provides ready-to-use philosophical questions organized into 10 categories. Each topic has been carefully selected to work well with the Socratic method, encouraging exploration through questioning rather than assertion.

## Topic Categories

### 1. Classic Philosophy (10 topics)

Timeless questions from the Western philosophical tradition.

**Topics:**
1. **What is justice?**
   - Explores fairness, moral rightness, and social order
   - Classic question from Plato's Republic
   - Perfect for examining social and political philosophy

2. **What is the good life?**
   - Ancient Greek concept of eudaimonia
   - Examines happiness, virtue, and fulfillment
   - Connects ethics with personal development

3. **Can we know anything for certain?**
   - Epistemological inquiry into knowledge
   - Questions the foundations of certainty
   - Explores skepticism and justified belief

4. **What is virtue?**
   - Moral excellence and character development
   - Central to Aristotelian ethics
   - Examines what makes a person good

5. **Is knowledge power?**
   - Relationship between knowledge and authority
   - Francis Bacon's famous proposition
   - Questions the use and abuse of knowledge

6. **What is truth?**
   - Nature of truth and truthfulness
   - Correspondence, coherence, pragmatic theories
   - Foundation of epistemology

7. **What is beauty?**
   - Aesthetics and aesthetic experience
   - Subjective vs. objective beauty
   - Art, nature, and perception

8. **Can virtue be taught?**
   - Socratic question about moral education
   - Nature vs. nurture in ethics
   - Role of education in character development

9. **What is wisdom?**
   - Practical judgment and understanding
   - Difference between knowledge and wisdom
   - Socratic wisdom: knowing what you don't know

10. **What is the nature of reality?**
    - Metaphysical inquiry into existence
    - Appearance vs. reality
    - Fundamental structure of the universe

**Best For:** Students of philosophy, classical education, foundational exploration

## Using the Topic Library

### In the Web Interface

**Accessing Topics:**
1. Open the Socratic Sofa web interface
2. Look for the dropdown labeled "üìö Topic Library"
3. Click to expand and browse all topics
4. Topics are formatted as `[Category] Topic`
5. Select any topic to use it for your dialogue

**Special Option:**
- **"‚ú® Let AI choose"**: Default option that allows the AI to select an appropriate topic

### In the CLI

Topics are loaded from `src/socratic_sofa/topics.yaml`. To use specific topics:

**Method 1: Modify main.py**
```python
inputs = {
    'topic': 'What is justice?',  # Any topic from library
    'current_year': str(datetime.now().year)
}
```

**Method 2: Programmatic Access**
```python
import yaml
from pathlib import Path

def load_topics():
    topics_file = Path("src/socratic_sofa/topics.yaml")
    with open(topics_file, 'r') as f:
        return yaml.safe_load(f)

topics = load_topics()
ethics_topics = topics['ethics']['topics']
print(ethics_topics)  # List of 10 ethics topics
```

## Customization

### Modifying the Library

The topic library is stored in `src/socratic_sofa/topics.yaml`:

**Format:**
```yaml
category_key:
  name: "Display Name"
  topics:
    - "Topic question 1"
    - "Topic question 2"
    - "Topic question 3"
```

**Adding Topics:**
```yaml
your_category:
  name: "Your Category Name"
  topics:
    - "Your question 1?"
    - "Your question 2?"
    - "Your question 3?"
```

**Tips for Adding Topics:**
- Keep questions clear and open-ended
- Avoid yes/no questions
- Ensure philosophical depth
- Test with Socratic method in mind
- Consider multiple perspectives

### Creating Custom Categories

**Example: Environmental Ethics**
```yaml
environment:
  name: "Environmental Ethics"
  topics:
    - "Do future generations have rights?"
    - "Is nature intrinsically valuable?"
    - "Do animals have moral status?"
    - "Is sustainable development possible?"
    - "What is our duty to the planet?"
```

**Example: Digital Life**
```yaml
digital:

  topics:
    - "Is online identity real identity?"
    - "Do we own our digital selves?"
    - "Is the internet a public space?"
    - "Should algorithms be transparent?"
    - "What is digital literacy?"
```
